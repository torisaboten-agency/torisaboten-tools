## 🎯 核心工具

### 🎲 线下抽奖小助手 (`ichibankuji/`)

**专业的活动抽奖工具**，为线下聚会、商铺活动、粉丝见面会等场景提供公平透明的抽奖体验。

#### ✨ 核心功能
- **🎯 双重抽奖模式**
  - 📦 **一番赏模式**：经典盒抽体验，支持多箱抽奖、LAST赏特殊奖励
  - 🎰 **概率抽奖**：灵活设置奖项概率，适合各种活动场景
- **💎 高级特性**
  - 🔄 **智能撤销**：支持跨箱撤销，灵活处理特殊情况
  - 📊 **数据管理**：完整的抽奖记录和结果统计
  - 💾 **本地存储**：数据安全可靠，支持快照导入导出
  - 🔍 **搜索筛选**：快速查找历史抽奖记录
- **🎨 用户体验**
  - 📱 **响应式设计**：完美适配手机、平板、电脑
  - 🌙 **现代UI**：简洁优雅的界面设计
  - ⚡ **即开即用**：无需安装，打开即可使用

#### 🎪 适用场景
- **商铺活动**：新店开业、节日促销、客户回馈
- **粉丝聚会**：生日会抽奖、应援活动奖品分发
- **企业活动**：年会抽奖、团建活动、员工福利
- **社区活动**：社区聚会、志愿活动、公益抽奖

### 🎭 参战计划作成工具 (`sansen_planner_vue/`)

**偶活粉丝专属的参战规划神器**，帮您合理安排演出时间，避免时间冲突，制作专业的参战计划表。

#### ✨ 核心功能
- **📅 多规划器管理**
  - 🏷️ 独立规划器：不同活动单独管理，井然有序
  - 🔄 智能切换：快速在多个活动间切换
  - 📂 数据隔离：各规划器数据完全独立
- **🎪 活动管理系统**
  - 🎭 **团体管理**：添加喜爱的偶像团体和个人
  - 📍 **活动配置**：设置活动名称、地点、日期
  - ⏰ **时间规划**：精确设置Live演出和特典会时间
- **📊 可视化甘特图**
  - 🎨 **双色时间条**：蓝色Live演出 + 绿色特典会，一目了然
  - 📈 **实时预览**：修改即时显示，所见即所得
  - 🎯 **冲突检测**：自动识别时间冲突，合理安排行程
- **💾 数据管理**
  - 📷 **高清导出**：一键生成PNG图片，保存到相册分享
  - ☁️ **本地存储**：数据本地保存，隐私安全
  - 📱 **跨设备同步**：支持数据导入导出，多设备使用

#### 🎯 适用场景
- **🏟️ 大型偶像节**：AKB48握手会、乃木坂46全国握手会等多团体活动
- **🎤 巡回演唱会**：单日多场次Live + 特典会完美规划
- **🤝 握手会活动**：合理安排Live观看和握手时间
- **🎪 粉丝见面会**：充分利用时间，不错过任何精彩瞬间

#### 🚀 技术亮点
- **⚡ Vue 3 + TypeScript**：现代化前端技术栈
- **🎨 响应式设计**：PC端左右分栏，移动端上下布局
- **📱 移动优先**：触控友好，手机上也能轻松操作
- **🔧 模块化架构**：代码结构清晰，维护性强

## 🚀 快速开始

### 🌐 在线使用（推荐）
直接访问 [tools.torisaboten.com](https://tools.torisaboten.com) 即可立即开始使用，无需安装任何软件。

### 💻 本地运行
```bash
# 克隆项目
git clone https://github.com/username/torisaboten-tools.git
cd torisaboten-tools

# 线下抽奖小助手（无需构建）
# 直接在浏览器中打开 ichibankuji/index.html

# 参战计划作成工具（需要构建）
cd sansen_planner_vue
npm install
npm run dev
```
