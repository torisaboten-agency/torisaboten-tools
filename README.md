# 实用工具集合

这是一个包含多个实用工具的项目集合，涵盖抽奖、决策、规划等日常场景。

## 🛠️ 工具列表

### 1. 抽奖工具 (`ichibankuji/`)
线下抽奖小助手，支持一番赏和概率抽奖两种模式。

**功能特点：**
- 🎯 双重抽奖模式（一番赏/概率抽奖）
- 📦 多箱设置和LAST赏机制
- 💾 本地数据存储和快照导出
- 📱 响应式设计，支持桌面和移动端

**使用方法：**
```bash
# 在ichibankuji目录下打开index.html即可使用
```
### 2. 偶活参战规划小助手 (``sansen_planner_vue/`)
基于Vue 3 + TypeScript，专为偶像活动粉丝设计的参战规划工具，生成可视化甘特图。

**功能特点：**
- 📅 多规划器管理，支持不同活动独立规划
- 🎭 活动和团体管理，灵活配置演出安排
- ⏰ Live演出和特典会时间段分别设置
- 📊 实时甘特图生成，按活动分组显示
- 🎨 双色时间条（蓝色Live + 绿色特典）
- 📷 一键导出高清PNG图片到相册
- 📱 响应式设计，PC端左右分栏，移动端上下布局
- ✅ 完整的分界线显示，确保时间轴视觉连贯性
- 🎯 紧凑的时间轴设计，减少空间占用
- 🔧 智能的最小宽度设置，避免显示错位
- 📤 系统日历导入功能，方便日程同步

**典型使用场景：**
- 大型偶像节多团体参战规划
- 握手会Live+特典时间安排
- 巡回演唱会单日多场次规划
- 粉丝见面会活动时间管理

### 3. 抛硬币工具 (`coin_flip/`)
简洁易用的在线抛硬币工具，帮助快速做决策。

**功能特点：**
- 🪙 流畅的硬币翻转动画，直观的视觉反馈
- ⚙️ 自定义正反面结果文本，适应各种决策场景
- 📊 完整历史记录和统计分析
- ⌨️ 键盘快捷键支持（空格键抛硬币）
- 🎨 统一银色硬币设计，简洁美观
- 📱 响应式设计，完美适配各种设备

**使用方法：**
```bash
# 在coin_flip目录下打开index.html即可使用
```

**典型使用场景：**
- 日常决策：选择困难时的快速裁决工具
- 娱乐游戏：聚会互动和运气测试
- 教育教学：概率演示和决策训练
- 团队协作：公平决策和打破僵局

## 🚀 开发

当前项目使用现代前端技术栈，支持多种开发方式。

**技术栈：**
- HTML5 + CSS3 + JavaScript ES6+
- Vue 3 + TypeScript（现代版本）
- Canvas 2D API（甘特图绘制）
- LocalStorage（本地数据存储）
- 响应式设计（支持桌面和移动端）

**运行方式：**
```bash
# 静态工具（一番赏、抛硬币、偶活规划原版）
# 直接在浏览器中打开HTML文件即可

# Vue版本工具
cd sansen_planner_vue/
npm install
npm run dev

# Vuetify版本工具
cd sansen_planner_vuetify/
npm install
npm run dev
```

## 🌟 特色

### 统一设计语言
- 所有工具保持一致的界面风格和交互体验
- 共享样式文件确保视觉统一性
- 标准化的组件和布局模式

### 本地化存储
- 完全基于浏览器LocalStorage
- 不同用户数据完全隔离
- 无需服务器，完全离线可用

### 响应式设计
- 完美适配桌面端和移动端
- 智能布局切换（左右分栏 ↔ 上下堆叠）
- 触控友好的操作界面

### 数据导出
- 支持多种格式导出（快照JSON、记录CSV、图片PNG）
- 便于数据备份和分享
- 高质量图片适合保存到相册
