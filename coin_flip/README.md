# 抛硬币工具

一个精美的在线抛硬币工具，支持自定义选项和历史记录。

## 🎯 功能特点

### 核心功能
- **🪙 3D硬币翻转动画** - 真实的硬币翻转效果，让抉择更有仪式感
- **⚙️ 自定义选项** - 可自由设置正面和反面的结果文本
- **📊 历史记录** - 完整记录每次抛硬币的结果和时间
- **📈 统计分析** - 实时显示总次数和正反面分布

### 交互体验
- **⌨️ 键盘快捷键** - 按空格键快速抛硬币
- **💾 自动保存** - 设置和历史记录自动保存到本地
- **🎨 视觉反馈** - 丰富的动画效果和色彩区分
- **📱 响应式设计** - 完美适配桌面和移动设备

## 🚀 使用方法

### 基本操作
1. **设置选项**：在顶部输入框中自定义正面和反面的结果文本
2. **抛硬币**：点击"🪙 抛硬币"按钮或按空格键
3. **查看结果**：等待翻转动画完成，查看结果显示
4. **历史记录**：在底部查看所有抛硬币的历史记录

### 自定义选项示例
- **决策类**：正面设为"同意"，反面设为"拒绝"
- **方向类**：正面设为"左边"，反面设为"右边"
- **行动类**：正面设为"去做"，反面设为"再等等"
- **选择类**：正面设为"选项A"，反面设为"选项B"

### 快捷操作
- **空格键**：快速抛硬币
- **清空历史**：点击"🗑️ 清空历史"按钮清除所有记录

## 🎨 设计特色

### 视觉效果
- **金色硬币设计** - 采用金色渐变和阴影效果，质感十足
- **3D翻转动画** - 真实的空间旋转效果，增强沉浸感
- **色彩区分** - 正面绿色，反面蓝色，便于快速识别

### 动画细节
- **翻转轨迹** - 硬币在X轴和Y轴上同时旋转，模拟真实抛硬币
- **缓动函数** - 使用cubic-bezier缓动，让动画更自然
- **动画时长** - 1.5秒的翻转时间，既不过长也不过短

### 界面布局
- **卡片式设计** - 每个功能区域独立成卡片，层次分明
- **渐变背景** - 柔和的渐变色彩，提升视觉体验
- **圆角设计** - 现代化的圆角元素，界面友好

## 📊 数据管理

### 本地存储
- **自动保存** - 所有设置和历史记录实时保存到浏览器本地存储
- **持久化** - 刷新页面后数据不丢失
- **隐私保护** - 所有数据仅存储在本地，不上传到服务器

### 历史记录功能
- **详细记录** - 记录结果、时间戳、当时的正反面设置

## 🔧 技术实现

### 前端技术
- **Vue 3** - 响应式数据绑定和组件化开发
- **CSS3 动画** - 硬币的3D翻转效果
- **Local Storage** - 本地数据持久化存储
- **响应式设计** - 适配各种屏幕尺寸

### 动画原理
- **CSS 3D Transform** - 使用rotateX和rotateY实现空间旋转
- **Perspective** - 设置透视距离，增强3D效果
- **Keyframes** - 定义翻转动画的关键帧
- **Transform-style** - 保持3D变换空间

### 随机算法
- **Math.random()** - 使用JavaScript内置随机数生成器
- **50/50概率** - 严格的二分概率分布
- **即时生成** - 点击时实时生成结果，保证随机性
